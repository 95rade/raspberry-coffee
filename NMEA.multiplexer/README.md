# NMEA Multiplexer
Any input (File, Serial, TCP, UDP, RMI, WebSocket...), any output (File, Serial, TCP, UDP, RMI, WebSockets...), and a REST API on top of that.

### Includes
- NMEA Parser
- NMEA String generator
- Serial port Reader / Writer
- TCP Reader / Writer
- UDP Reader / Writer
- WebSocket client
